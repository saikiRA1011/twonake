# twonake

## 概要
2匹のヘビを同時に操作するヘビゲームです。1人か2人でのプレイを想定しています。

緑色のヘビはAキー、Dキーで、赤色のヘビはJキー、Lキーで操作します。それぞれのヘビは自分と同じ色のフルーツを食べることで成長し、スコアを獲得できます。
ただし、違う色のフルーツや、壁、自分や他のヘビの体にぶつかるとゲームオーバーです。

ランキング機能を搭載しており、4位までのスコアを確認することができます。
